📋 Custom Data Structures Reference

For Linked List and Tree problems, the platform automatically parses the standard JSON array input and converts it into custom Python objects. Your solution should operate directly on these nodes, and return the head (for Linked List) or root (for Trees) node.

Singly Linked List Node (ListNode)

class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

Binary Tree Node (TreeNode)

class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

🗺️ The 13 Key DSA Coding Patterns

  • Arrays & Hashing: Hash map lookup, frequency maps, prefix products, prefix sums.
  • Two Pointers: Left/Right boundaries moving inwards, fast/slow boundaries.
  • Sliding Window: Dynamic or fixed-sized subsegments tracking, shrinking boundaries.
  • Stack & Queue: FIFO / LIFO orderings, monotonic stacks.
  • Binary Search: Searching sorted domains, binary searching answer space.
  • Linked List: Pointer manipulation, reversals, merging.
  • Trees & BST: Recursion, depth-first traversal (pre/in/post-order), level-order traversal.
  • Heap / Priority Queue: Realtime min/max tracking, merging streams, top k items.
  • Backtracking: Combinatorial generation, pruning search space, N-Queens.
  • Graphs & BFS/DFS: Traversing adjacency grids, topological sorting, shortest paths.
  • Greedy: Locally optimal choices making global solutions, intervals merging.
  • Dynamic Programming: Memoized recursion, tabular subproblems.
  • Advanced Patterns: Prefix Trees (Tries), Bit Manipulation tricks.
